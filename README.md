# Boslaantje Doen — website

Broncode voor www.boslaantjedoen.nl (snackbar, catering, zakelijke lunch & verhuur, Veenendaal).

## Structuur

- `index.html` — homepage
- `catering.html` — cateringpagina (BBQ, buffet, hapjes, offerte-aanvraag)
- `images/` — map voor afbeeldingen (nu leeg — zie "Ontbrekend" hieronder)

## Ontbrekend / nog toevoegen

- `vers-en-belegd.html` en `vers-en-belegd-zakelijke-lunch.html` — worden vanaf beide pagina's gelinkt maar zijn nog niet aangeleverd.
- `images/favicon-32.png` — wordt gerefereerd in `catering.html` maar bestaat nog niet in deze repo.
- Overige afbeeldingen die in de uiteindelijke site gebruikt worden.

## Lokaal bekijken

Open `index.html` gewoon in de browser, geen buildstap nodig (alles is losse HTML/CSS, fonts zijn inline als base64).
